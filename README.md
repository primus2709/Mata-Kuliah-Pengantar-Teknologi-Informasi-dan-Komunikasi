# Pengantar Teknologi Informasi dan Komunikasi (PTIK)

Repository ini berisi kumpulan **tugas, latihan, dan proyek** dari mata kuliah **Pengantar Teknologi Informasi dan Komunikasi (PTIK)**.  
Tujuannya adalah untuk mendokumentasikan proses pembelajaran dan penerapan dasar-dasar teknologi informasi dalam kehidupan akademik maupun profesional.
